# Fake Degree & Certificate Detection System

This project detects whether an uploaded academic certificate is genuine or fake using OCR and verification logic.

## Features
- Upload certificate (PDF/Image)
- OCR-based text extraction
- Verification against sample database
- Confidence score output
- Fake / Genuine classification

## Tech Stack
- FastAPI
- Python
- Tesseract OCR
- HTML/CSS/JS

## How It Works
1. User uploads certificate
2. OCR extracts text
3. Details are cross-verified
4. Confidence score is generated

## Result
The system outputs authenticity with confidence percentage.
